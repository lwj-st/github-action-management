## 🚀 立即开始使用

### 第一步：安装依赖

```bash
cd /opt/data/workspace/github-action-management
npm install
```

### 第二步：启动应用

```bash
npm run tauri dev
```

### 第三步：配置 Token

1. 参考 [`TOKEN_SETUP.md`](TOKEN_SETUP.md) 创建 PAT Token
2. 在应用中添加账户
3. 开始管理你的 GitHub Actions
